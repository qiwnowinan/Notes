# 记录两者的特别之处

## for

在for循环里面  
C只能`int  i; for(i = 0;)`
C++最好`for(int i = 0)`

## bool类型

cpp有bool(true/false)类型，C没有  

## 数组初始化

`int arr5{}; // C++特有，所有元素初始化为0`
`char a{}; //全部是'\0'`

C需要`int arr3 = {0};`

计算数组长度`int len = sizeof(arr) / sizeof(arr[0]);`

## 结构体

`struct Student stu1 = {10,0,1,10086}; //C里面要有(struct) ,C++可以没有`
