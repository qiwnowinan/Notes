### 整型
|类型|关键字|典型大小|范围|
|---|---|---|---|
短整型|short|2字节|-32,768 ~ 32,767|
整型|int|4字节|-2,147,483,648 ~ 2,147,483,647|
长整型|long|4/8字节|取决于系统|
长长整型|long long|8字节|-9.2×10¹⁸ ~ 9.2×10¹⁸|

大于long long只能用字符串了，虽然double更大，单精度太低

### 浮点型
类型|关键字|精度|大小|有效位数|
|---|---|---|---|---|
单精度|float|6-7位小数|4字节|~7位|
双精度|double|15-16位小数|8字节|~16位|

`float f = 3.14159f; //float字面量加f后缀`  
3.14默认是double  
有效位数是数字中所有可靠的数字（从第一个非零数字到最后一个数字）  
精度是小数点后能准确表示多少位数字  

### 科学技术法

## 优先级
| 优先级 | 运算符 | 描述 | 结合性 |
| :--- | :--- | :--- | :--- |
| 1 (最高) | `::` | 作用域解析 (C++特有) | 左 → 右 |
| 2 | `. -> [] ()` 后缀`++ --` | 成员访问、下标、函数调用、后缀自增/自减 | 左 → 右 |
| 3 | `++ -- ~ ! + - * & sizeof` (一元) | 前缀自增/自减、逻辑非、一元正负、解引用、取地址等 | 右 → 左 |
| 4 | `.* ->*` | 成员指针访问 (C++特有) | 左 → 右 |
| 5 | `* / %` | 乘、除、取模 | 左 → 右 |
| 6 | `+ -` | 加、减 | 左 → 右 |
| 7 | `<< >>` | 左移、右移 | 左 → 右 |
| 8 | `< <= > >=` | 关系比较 | 左 → 右 |
| 9 | `== !=` | 相等比较 | 左 → 右 |
| 10 | `&` | 按位与 | 左 → 右 |
| 11 | `^` | 按位异或 | 左 → 右 |
| 12 | `\|` | 按位或 | 左 → 右 |
| 13 | `&&` | 逻辑与 | 左 → 右 |
| 14 | `\|\|` | 逻辑或 | 左 → 右 |
| 15 | `?:` | 条件 (三元) 运算符 | 右 → 左 |
| 16 | `= += -= *= /= %= ...` | 赋值及复合赋值 | 右 → 左 |
| 17 (最低) | `,` | 逗号运算符 | 左 → 右 |

## 数组

### 初始化

```cpp
int arr1[5] = {1, 2, 3, 4, 5}; // 完全初始化
```

## 排序

### 桶排序
4 7 2 3 8 9  
有10个桶，编号1-10  
有4，则4号桶+1  
然后遍历编号桶  

### 冒泡排序
34 23 78 176 453  
从第1个开始，两个比大小，大的在前，直到末尾  
此时最大在第1个位置
再从第2个开始，进行一样的操作，直到结束  
此时第二大的在第2个位置  
···

### 选择排序
找最大，放结尾  
找第二大，放倒数第二  
···
