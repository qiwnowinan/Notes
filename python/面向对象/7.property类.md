# property

为了防止出现类的数据出现`age = -2`之类的错误变量赋值，我们可以设置一个set_age函数和get_age函数进行获得值和改变值，但是为了在保护数据安全的同时，保持代码的简洁易用，引入这个类

## 直接用（用装饰器更好）

```py
class Student:
    def __init__(self, name, age):
        self.name = name
        self.__age = age

    def get_age(self):
        return self.__age

    def set_age(self, age):
        if age > 0 and age < 120:
            self.__age = age
        else:
            raise Exception

    # 使用property函数创建属性，可以直接访问age属性，背后调用get_age和set_age方法
    age = property(fget=get_age, fset=set_age)


s1 = Student("Jack", 18)
s1.age = 3  # 直接访问age属性，实际上调用了set_age方法
print(s1.age)  # 直接访问age属性，实际上调用了get_age方法

# 当执行 s1.age = 3 时，Python 实际上是在调用 s1.set_age(3)。
# 当执行 print(s1.age) 时，Python 实际上是在调用 s1.get_age()。

# 原本的方法也行
s1.set_age(5)
print(s1.age)
```

## 装饰器版

```py
class Student:
    def __init__(self, name, age):
        self.name = name
        self.__age = age

    @property  # 当加上@property装饰器时，age方法就变成了一个属性，可以直接通过s1.age来访问，而不需要加括号
    def age(self):  # 当访问age属性时，调用这个方法
        return self.__age

    # 函数的名字即（age）必须和属性的名字（age）一样（三个地方名字一样），否则会报错AttributeError: can't set attribute
    @age.setter  # 当加上@age.setter装饰器时，age方法就变成了一个属性的setter方法，可以通过s1.age = 20来设置age属性的值
    def age(self, age):  # 当设置age属性时，调用这个方法
        if age > 0 and age < 120:
            self.__age = age
        else:
            raise Exception


s1 = Student("Jack", 18)
s1.age = 20
print(s1.age)
```

## 只读property 和 删除property

```py
class Square:
    def __init__(self, width):
        self.__width = width
        self.__area = None # 为了了实现area的缓存，初始值为None表示未计算过，否则每次访问area属性时都要重新计算面积，浪费性能

    @property  
    def width(self):  
        return self.__width

    
    @width.setter  
    def width(self, width):  
        if width > 0:
            self.__width = width
            self.__area = None  # Reset area when width changes
        else:
            raise Exception

    @property # 没有area.setter,所以是只读的
    def area(self):
        if self.__area is None:
            self.__area = self.__width * self.__width
        return self.__area

    @area.deleter
    def area(self):  # 在del此属性时，自动调用此方法
        self.__area = None  # 删除area属性时，重置为None表示未计算过

s1 = Square(5)
print(s1.area)  # 25
del s1.area  # 删除area属性，重置为None表示未计算过
print(s1.area)  # 25，重新计算面积
```
