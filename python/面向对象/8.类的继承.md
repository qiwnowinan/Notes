# 继承

## 单继承

```py
class Person:
    def __init__(self, name):
        self.name = name

class Student(Person): # Person里的属性和方法，Student全有
    def __init__(self):
        super().__init__("Jack") # super()是父类，执行父类初始化
        self.school = "abc" 

isinstance(student, Student) # True
isinstance(student, Person) # True
isinstance(person, Student) # False
issubclass(Student, Person) # True
```

## 方法/属性重写

在子类里重新定义父类方法/属性，调用从子类开始找，找不到找父类

```py
class Person:
    index = 1

    def __init__(self, name):
        self.name = name

    def say(self):
        print(f"Hello {self.name}")

    def print_index(self):
        print(self.__class__.index)


class Student(Person):
    index = 2

    def __init__(self):
        super().__init__("Jack")
        self.school = "abc"

    def say(self):
        print(f"Hello {self.school}")


s1 = Student()
s1.print_index() # 2
```

## 多继承

```py
class Parent1:
    def hello(self):
        print("Parent 1")

class Parent2:
    def hello(self):
        print("Parent 2")

class Child(Parent1,Parent2): # 从左往右继承
    pass

child = Child()
child.hello() # Parent 1 按继承的顺序来弄

# 可以直接在Child重写hello
class Child(Parent1,Parent2): # 从左往右继承
    def hello(self):
        super().hello() # 也是按继承的顺序来弄
        Parent2.hello(self) # 这不是静态方法，所以要加self
```
