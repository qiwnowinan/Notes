# 类变量

属于类本身这个对象的属性，该类的对象共享此变量  
类变量全在__dict__(类变量)这个字典中，但不要直接通过这个字典改

## 默认定义的类变量

```py
class Student:
    pass

print(Student.__name__) # 这个类属性是类的名字（默认定义的）
```

## 自定义

```py
class Student:
    student_count = 0

# 调用
print(Student.student_count) # 方法1,如果没定义会报错
print(getattr(Student,"student_count","10")) # 方法2，如果没定义，不报错，返回10

# 更改/增加
Student.student_count = 89 # 直接改
setattr(Student,"student_count","10") # 用函数

# 删除
del Student.student_count
```

## 在实例函数里面的调用

`self.__class__.变量名 # self 代表实例，而 self.__class__ 则指向该实例所属的类。`
