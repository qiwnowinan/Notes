# 定制和拓展枚举

## 定制str

```py
class Gender(Enum):
    MALE = 1
    FEMALE = 2

    def __str__(self): # 这里的self是像Gender.MALE这样的对象
        return f"{self.name}:{self.value}"
```

## 定制eq

`Status.SUCCESS == Status.OK`可直接比较  
但如果要`Status.SUCCESS == 1`就要定制__eq__

```py
def __eq__(self,other):
    if isinstance(other, int):
        return self.value == other

    if isinstance(other, str):
        return self.name == other.upper()  # name都是大写

    if isinstance(other, Gender):
        return self is other  

    return False
```

## 定制lt

```py
from functools import total_ordering

@total_ordering # lt是<的情况，有这个装饰器会自动补全其他情况
class Status(Enum):
    SUCCESS = 1 
    OK = 2
    FAIL = 3
    WRONG = 4

def __lt__(self, other):
    if isinstance(other, int):
        return self.value < other
    return False
```
