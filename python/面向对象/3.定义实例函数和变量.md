# 实例函数和变量

## 实例函数

```py
class Student:
    def say_hello(self,msg):
        print(f"Hello,{msg})

s1 = Student()
s1.say_hello("Jack") # self在这里就是s1
# 通过s1.方法来调用实例方法
```

## 实例对象

1. create a physical object
2. call __init__ to initialize this object

```py
class Student:
    def __init__(self,name):  # 用__init__定义实例变量
        self.name = name      # 用于初始化对象的实例函数

    def say_hello(self,msg):
        print(f"Hello,{msg}, {self.name}")
        # 在实例函数里通self访问实例变量

s1 = Student("Jack") # 此时__init__执行
