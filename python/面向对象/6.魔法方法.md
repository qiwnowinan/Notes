# 魔法方法

## str

给用户看,在str(),print()自动调用

```py
def __str__(self):
    return f"用户: {self.name}"
```

## repr

给开发者看,在repr()自动调用

```py
    def __repr__(self):
        return f"User('{self.name}', {self.age})"
```

## eq

在...==...时自动调用

```py
def __eq__(self, other):
        if not isinstance(other, User):
            return False
        return self.name == other.name
```

## hash

作用：定义 hash(obj) 的值，决定了对象能否作为字典的键或放入集合。  
铁律：如果重写了 __eq__，必须重写 __hash__。  
如果两个对象 a == b，那么 hash(a) 必须等于 hash(b)。  
如果对象是可变的（内容会变），通常建议把 __hash__ 设为 None（即不可哈希），防止它作为字典键后内容变了，导致字典找不到它。  

可以把关键属性打包成元组

```py
def __hash__(self):
        # 把关键属性打包成元组 (name, age)
        # 元组是可哈希的，且内容一样哈希值就一样
        return hash((self.name, self.age))
```

## bool

在if obj:, bool(obj)时调用

```py
def __bool__(self):
        return self.is_active
```

## del

对象被垃圾回收（销毁）前调用
