# 私有属性和函数

防止在类的外面被使用，主要通过加_前缀来限制

## 具体做法

    def __init__(self, name):
        self._name = name # 加一个_
        self.__name = name # 加两个_

    def say_hello(self,msg):
        print(f"Hello,{msg}, {self._name}")
