# 类方法和静态方法

## 类方法

特征：必须加 @classmethod 装饰器，第一个参数必须是 cls（代表类本身）。  
功能：它可以访问和修改类变量，但不能访问实例变量  
调用方式：类名.方法() 或 实例.方法()（推荐用类名）。  
核心场景：作为“备用构造函数”  
这是类方法最高级的用法。Python 的 __init__ 只能有一种初始化方式，但可以用类方法创建多种“快捷入口”。

```py
class Student:
    school = "abc"

    @classmethod
    def hello(cls): # cls就是类本身
        print(f"Hello {cls.__name__}")
        print(f"Hello {cls.school}")
```

### 备用构造函数实例

```py
class Student:
    school = "阳光小学"

    def __init__(self, name, age):
        self.name = name
        self.age = age

    # 普通实例方法
    def info(self):
        print(f"{self.name}, {self.age}岁")

    # --- 类方法 ---
    @classmethod
    def create_from_birth_year(cls, name, birth_year):
        # 1. 计算年龄
        age = 2026 - birth_year
        # 2. 返回一个实例 (相当于调用了 __init__)
        return cls(name, age)

    @classmethod
    def change_school(cls, new_school):
        # 修改类变量，影响所有实例
        cls.school = new_school

# 使用
# 1. 使用备用构造函数
s1 = Student.create_from_birth_year("小明", 2016) 
s1.info() # 输出: 小明, 10岁

# 2. 修改全校的名字
Student.change_school("希望小学")
```

## 静态方法

就像是借住在类里面的普通函数，是为了解决代码组织、逻辑归属和命名空间管理的问题  
特征：必须加 @staticmethod 装饰器，不需要 self 也不需要 cls 参数。  
功能：它既不访问实例变量，也不访问类变量。它完全独立，只是逻辑上属于这个类。  
调用方式：类名.方法()（推荐）或 实例.方法()。

```py
class MathUtils:
    # --- 静态方法 ---
    @staticmethod
    def add(a, b):
        return a + b

print(MathUtils.add(3, 5))        # 输出: 8
```
