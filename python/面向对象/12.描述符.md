# 描述符

first_name和last_name逻辑可能一样，为了省去多余的代码  
如果要在 10 个类中实现“年龄必须是正整数”的逻辑，需要写 10 遍 @property 或者 getter/setter 方法。  
使用描述符后，只需要写一次 PositiveInt 类，然后在 10 个类中直接引用即可。

```py
# 此模块记为Student1.py
import Student2
class Student1:
    def __init__(self, first_name, last_name):
        self.__first_name = first_name
        self.__last_name = last_name

    @property  
    def first_name(self):  
        return self.__first_name

    
    @first_name.setter
    def first_name(self, first_name):  
        if not isinstance(first_name, str):
            raise Exception

        if len(first_name) == 0:
            raise Exception

        self.__first_name = first_name
```

## 定义

描述符本质上是一个对象，只要它的类实现了 __get__、__set__ 或 __delete__ 这三个特殊方法中的至少一个，它就是一个描述符。  

是一个特殊的类，要求实现如下一些类  
__set_name__(self, owner, name) # 一定实现，剩下3个至少实现一个
__get__(self, instance, owner)
__set__(self, instance, value)
__delete(self, instance)  

```py
# 此模块记为Student2.py
class Student2:
    first_name = RequiredString(True)  # RequiredString()就是描述符
    last_name = RequiredString(True)  # 此时__set_name__和__init__已经调用(owner = Student2, name = last_name)
    password = RequiredString(False)

student_2 = Student2()        # 当指向描述符时
student_2.first_name = "Jack" # 本质上是RequiredString对象，进行赋值操作是把调用了RequiredString的__set__方法
# first_name.__set__(student_2,"Jack")  分别给instance, value
student_2.last_name = "Smith"
student_3 = Student2()
student_3.first_name = "Rose"
print(student_2.first_name)
print(student_2.last_name)
print(student_3.first_name)
```

```py
# 此模块记为RequiredString.py
class RequiredString:
    def __init__(self, trim: bool):
        self.trim = trim

    def __set_name__(self, owner, name):
        self.__name = name

    def __set__(self, instance, value):
        if not isinstance(value, str):
            raise Exception

        if self.trim:
            value = value.strip()

        if len(value) == 0:
            raise Exception

        instance.__dict__[self.__name] = value

    def __get__(self, instance, owner):
        return instance.__dict__[self.__name]
```
