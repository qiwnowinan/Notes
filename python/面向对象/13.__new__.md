# new

所有类的父类是object类  
object类里面有`object.__new__(self, *args, **kwards)`  
我们可以重写这个方法

## 创建对象的过程

`person = Person("Jack")`

1. person = object.__new__(Person, "Jack")
2. person.__init__("Jack")

## 实例

实例1

```py
class SquareNumber(int):
    def __new__(cls, value: int):
        return super().__new__(cls, value ** 2) # 因为继承了int，所以直接调用父类的__new__

num = SquareNumber(4)
print(num)  # 16
print(type(num)) 
print(isinstance(num, int)) # True
```

实例2

```py
class Student:
    # 这与__init__很像
    def __new__(cls, first_name, last_name):
        obj = super().__new__(cls)
        obj.first_name = first_name
        obj.last_name = last_name
        return obj # 此时不需要__init__
```
