# 枚举

它不是简单的变量，而是一种自定义的数据类型。就像 int 代表整数，str 代表字符串，Gender（你的枚举）代表“性别”这一种特定的类型。  
给人看时代码像英语句子一样通顺。  
给机器看时强制约束范围，防止非法值。  
给IDE看：支持自动补全，防止拼写错误。  

## 用法

```py
from enum import Enum

class Gender(Enum):
    MALE = 1
    FEMALE = 2

print(Gender.MALE)  # Gender.MALE
print(type(Gender.MALE))  # <enum 'Gender'>
print(Gender.MALE.value)  # 1
print(Gender.MALE.name)  # MALE

# 方法1
class Student:
    def __init__(self):
        self.gender = Gender.MALE

# 方法2
class Student:
    def __init__(self, gender: Gender):
        self.gender = gender

student1 = Student(Gender.MALE)

if student1.gender == Gender.MALE:
    print("学生1是男性")

s_gender = "FEMALE"
student.gender = Gender[s_gender]  # 将字符串转换为枚举成员
print(student.gender)  # Gender.FEMALE

i_gender = 1
student.gender = Gender(i_gender)  # 将整数转换为枚举成员
print(student.gender)  # Gender.MALE
```

## 遍历

```py
for i in Gender:
    print(i)
```

## 枚举别名

```py
class Status(Enum):
    SUCCESS = 1 # 只有一个是主要的，其他是别名
    OK = 1
    FAIL = 2
    WRONG = 2

for i in Status: # SUCCESS
    print(i)     # FAIL

print(Status.__members__) # 打印主要和别名
print(Status.SUCCESS == Status.OK) # True
print(Status.SUCCESS is Status.OK) # True
```

## 装饰器

```py
@enum.unique # 值重复会报错
class Status(Enum):
    SUCCESS = 1 # 只有一个是主要的，其他是别名
    OK = 1
    FAIL = 2
    WRONG = 2
```
