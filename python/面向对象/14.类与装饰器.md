# 类与装饰器

## 类装饰器

```py
class MyDecorator:
    def __init__(self, f):
        self.f = f

    def __call__(self, *args, **kwargs):
        print("start")
        result = self.f(*args, **kwargs)
        print("end")
        return result

@MyDecorator
def test(): # 当这个函数执行时，类的__call__会调用
    pass
```

### 带参数

```py
class ParamDecorator:
    def __init__(self, name):
        self.name = name

    def __call__(self, f):
        def wrap(*args, **kwargs):
            print("start")
            result = f(*args, **kwargs)
            print("end")
            return result
        return wrap


@ParamDecorator("Jack")
def test():
    pass
```

## 装饰类

```py
def cls_Decorator(cls):
    print("start class Decorator")
    def inner(*args, **kwargs):
        print("start")
        obj = cls(*args, **kwargs)
        print("end")
        return obj

    return inner


@cls_Decorator
class Person:
    def __init__(self, name):
        self.name = name
