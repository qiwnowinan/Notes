# 正则表达式

请开预览

- . 表示除换行符以外的任意字符
- at[ce] 表示atc/ate
- at[a-z] at+一个小写字符
- at[^a-z] at+一个除了小写字符外的任意字符
- [a-zA-Z1-3] 同理

---

- \d = [0-9]
- \D = [^0-9]
- \w = 字母，数字，下划线
- \W = 同理除外
- \s = 空白（空格/tab/换行/回车）
- \S = 除空白
- \\.和[.]= .(真正的.)

---

- ^a 表示以a开头的行
- t$ 表示以t结尾的行
- \bin 以in开头的单词
- in\b 以in结尾的单词
- \bin\b 单独in
- \Bin\B 单词中间in

---

## 量词

- a+ 1次/多次
- a* 0次/多次
- a？ 1次/0次
- a+ 1次/多次
- a{3} 3次
- a{3,5} 3-5次
- a{3,} 3次及以上

---

正则默认贪婪模式  
量词+? = 非贪婪  
at{3,}  
***attttttt***d 所有t都被匹配

at{3,}?  
***attttttt***d 前3个t被匹配

---
()会把里面的当成一个整体处理并捕获  

- (at)* atatatatat...
- (at){3} atatat
- (P|p) = [Pp]

捕获的内容可以再次引用  
`(\d{4})-(\d{1, 2})-(\d{1, 2})`  
2024-05-01  
可以用分组改成月日年的格式  
用`$2-$3-$1`

(?:\d{4}) 前面有?:表示未捕获分组  

`\b([a-z])[a-z]*\1\b`  \1表示引用第1个分组  
匹配如：  
alibaba  
tencent

---

## 前瞻和后顾

$100
$200
$abc

### 前瞻

匹配某些字符前面的内容

`\$(?=\d+)` 匹配数字前的$  

`\$(?!d+)` 匹配非数字前的$  

### 后顾

匹配某些字符后面的内容

`(?<=\$)\d+` 匹配$后面的数字  

`(?<!\$)\d+` 匹配非$后面的数字

## 在python里的使用

```py
import re
pattern = r"\d{2}"
source = "I'm 80 years old"

result = re.search(pattern, source) # 返回的是第一个成功匹配Match对象
if result:
    print(result.group())  # 如果找到了匹配项，group()方法将返回匹配的字符串。在这个例子中，匹配项是"80"，因此输出将是"80"。
# 如果没有找到匹配项，re.search()将返回None，因此在调用group()之前，最好先检查一下result是否为None，以避免引发AttributeError。

result = re.match(pattern, source) # 从字符串的开头进行匹配，也返回一个Match对象，如果不匹配返回None

result = re.fullmatch(pattern, source) # 匹配整个字符串

re.findall(pattern, source) # 返回一个list，包含字符串里符合的所有成员(非Match对象)

it = re.finditer(pattern, source) # 返回一个迭代器(Match对象)
# 占内存消耗小 
for i in it:
    print(i.group())


# 把手机号中间四位替换成 ****
# 这里用到了分组 ()，\1 代表第一组，\2 代表第二组
new_text = re.sub(r'(\d{3})-\d{4}-(\d{4})', r'\1-****-\2', text)
print(new_text)
# 输出: 我的手机号是 138-****-5678
```
