# Type Hints

## 变量注释

语法为 变量名: 类型 = 值。

```python
# 基本类型
name: str = "Alice"
age: int = 25
height: float = 1.75
is_student: bool = True
```

## 函数注释

语法为 def 函数名(参数: 类型, ...) -> 返回值类型:。

```python
# 有参数和返回值
def add(a: int, b: int) -> int:
    return a + b

# 无返回值，使用 None
def print_message(message: str) -> None:
    print(message)

# 带默认参数
def greet(name: str = "Guest") -> str:
    return f"Hello, {name}!"
```

## 容器类型

| 类型 | 写法 | 说明 |
| :--- | :--- | :--- |
| 列表 (List) | `numbers: list[int]` | 一个包含整数的列表 |
| 字典 (Dict) | `user: dict[str, int]` | 键为字符串，值为整数的字典 |
| 元组 (Tuple) | `person: tuple[str, int]` | 第一个元素是字符串，第二个是整数的元组 |
| 集合 (Set) | `hobbies: set[str]` | 一个包含字符串的集合 |

## 特殊类型

表示变量可以是多种类型中的一种。从 Python 3.10 开始，可以使用 | 符号作为简写。

```python
def process_id(id: int | str) -> None:
    print(id)
```
