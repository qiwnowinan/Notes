# 生成器

## 生成器函数

在 Python 中，生成器（Generator）可以理解为一个“能记住自己执行状态”的函数。  
它的核心特点是惰性求值，即不是一次性把所有数据都计算并存入内存，而是按需逐个地产生数据。这使得生成器在处理大规模数据或无限序列时，能够极大地节省内存，提升程序性能。  
生成器的核心是 yield 关键字。当一个函数中包含 yield 时，它就变成了一个生成器函数。  
调用生成器函数时：它并不会立即执行函数体，而是返回一个生成器对象。  
迭代生成器对象时(如for循环），函数体才开始执行。  
遇到 yield 时：函数会“暂停”执行，并将 yield 后面的值“产出”给调用者。
下一次迭代时：函数会从上次“暂停”的地方“恢复”执行，直到再次遇到 yield 或函数结束。  

```py
def squares(count):
    for i in count:
        yield i**2;

for i in squares(3): # 生成器函数执行了
    pass

g = squares(5)  # g是生成器对象
```

## 生成器表达式

跟列表推导式很像

```py
# 生成器表达式：创建一个生成器，按需计算
squares_gen = (x**2 for x in range(10))
```
