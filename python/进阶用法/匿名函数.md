# lambda

是一种“没有名字”的函数。当需要一个小函数，但专门用 def 定义一个正式函数又觉得太麻烦、太啰嗦时，lambda 就是最好的选择。常作为参数传递给其他高阶函数

## 语法

`lambda 参数列表: 表达式`  
lambda：定义函数的关键字  
参数列表：可以有 0 个、1 个或多个参数，用逗号分隔（和普通函数一样）。  
表达式：函数体只能包含一个表达式，不能写复杂的逻辑（如 if-else 分支语句、for 循环等），且表达式的计算结果会自动作为返回值，不需要写 return。  

## 与map配合使用

```py
numbers = [1, 2, 3, 4, 5]
# 需求：将列表中的每个数字平方
# lambda x: x**2 定义了平方的规则
squared = list(map(lambda x: x**2, numbers))
print(squared)
# 输出: [1, 4, 9, 16, 25]
```

## 赋值给变量的使用

可以给它起名字（f = lambda...），但在代码规范上，如果它有了名字，通常意味着它应该用 def 来写。

```py
# 将 lambda 函数赋值给变量 add
add = lambda x, y: x + y

# 现在可以像调用普通函数一样调用它
print(add(3, 5))  # 输出: 8
print(add(10, 20)) # 输出: 30
```
