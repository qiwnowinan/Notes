# 闭包

## 高阶函数

高阶函数就是满足以下任一条件的函数：
接收一个或多个函数作为参数。
返回一个函数作为结果。

## 闭包概念

```py
def f():
    message = 'hello'
    def inner():    # 内函数+外变量=闭包
        print(message)
    message = 'hi'
    return inner

g=f()  # f执行完，会把inner和message给g
g() # 会打印'hi'，因为闭包捕获的是变量引用，而不是值的拷贝，所以它去查看 message 变量当前的值。
```

变量也可以是外函数的参数，用于大量制造别的函数

```py
def create_multiplier(factor):
    # factor 是外部函数的参数，将被内部函数捕获
    def multiplier(x):
        return x * factor
    return multiplier
# 通常函数执行完，局部变量（如 factor）会被销毁。但因为 multiplier 闭包引用了它，Python 的垃圾回收机制不会回收它，让它“永生”了。

# 创建两个不同的函数
double = create_multiplier(2) # 创建一个乘以2的函数
triple = create_multiplier(3) # 创建一个乘以3的函数

print(double(5))  # 输出: 10 (5 * 2)
print(triple(5))  # 输出: 15 (5 * 3)
```

## nonlocal

如果要更改外变量的值，直接改会认为是在内函数里定义一个局部变量，需要nonlocal声明

```py
def make_counter():
    count = 0
    def increment():
        nonlocal count  # 必须声明，否则 count += 1 会报错
        count += 1
        return count
    return increment

counter = make_counter()
print(counter()) # 1
print(counter()) # 2
```
