# python入门基础语法  

包含list,tuple,set,dict  

## list  

list是有序的，可变的，元素多样的，最常用的方式是使用方括号 []创建list，可以使用 list() 函数将其他可迭代对象（如字符串、元组）转换成列表，用for循环遍历  

### 索引与切片

#### 索引

列表中的每个元素都可以通过Index来访问。  
正向索引：从 0 开始，从左到右。list[0] 是第一个元素。  
负向索引：从 -1 开始，从右到左。list[-1] 是最后一个元素。

#### 切片

不会修改原始序列，而是返回一个新的序列  
基本语法结构为：sequence[start:stop:step]  
start: 切片的起始索引（包含该位置的元素）。  
stop: 切片的结束索引（不包含该位置的元素）。  
step: 切片的步长，即每隔多少个元素取一个。  
核心规则：左闭右开

- list[:] 创建原列表的一个浅拷贝
- list[::-1] 将序列完全倒序

### 常用函数

#### 添加元素

- list.append(x): 在列表末尾添加一个元素 x。
- list.insert(i, x): 在指定索引 i 的位置插入一个元素 x。
- list.extend(iterable): 将另一个可迭代对象（如列表）的所有元素添加到当前列表末尾。

#### 删除元素

- list.remove(x): 删除列表中第一个值为 x 的元素。
- list.pop(i): 删除并返回指定索引 i 的元素。如果不指定 i，默认删除并返回最后一个元素。
- del list[i]: 删除指定索引 i 的元素。

#### 修改与排序

- list[index] = '...' : 修改
- list.sort(): 对列表进行原地排序（直接修改原列表）。
- sorted(list): 对列表排序并返回一个新列表，原列表不变。
- list.reverse(): 原地反转列表。

#### 其他常用操作

len(list): 获取列表的长度（元素个数）。  
x in list: 检查元素 x 是否在列表中，返回 True 或 False。  
list.index(x): 返回值为 x 的元素第一次出现的索引。
list.count(x): 统计值为 x 的元素在列表中出现的次数。

### 列表推导式

[表达式 for 变量 in 可迭代对象 if 条件]

| 场景 | 语法结构示例 |
| :--- | :--- |
| 纯转换 | `[x*2 for x in data]` |
| 纯筛选 | `[x for x in data if x > 0]` |
| 转换+筛选 | `[x*2 for x in data if x > 0]` |
| 条件分支 | `[x if x>0 else 0 for x in data]` |
| 嵌套循环 | `[(x,y) for x in A for y in B]` |

## tuple

元组 (Tuple) 和列表非常相似，都是有序的,但它是不可变的。元组支持索引和切片操作

### 解包和打包

#### 打包

```python
# 将三个值打包成一个元组
coordinates = 10, 20, 30
print(coordinates)      # 输出: (10, 20, 30)

# 即使加上括号，本质也是打包
point = (5, 15)
```

- 当用逗号分隔多个值并将它们赋给一个变量时，Python 会自动将它们打包成一个元组。

```python
def my_function(*args, **kwargs):
    print(f"位置参数打包成了元组: {args}")
    print(f"关键字参数打包成了字典: {kwargs}")

my_function(1, 2, 3, a=4, b=5)
# 输出:
# 位置参数打包成了元组: (1, 2, 3)
# 关键字参数打包成了字典: {'a': 4, 'b': 5}
```

- 在函数定义中，*和 **用于收集不定数量的参数，这也是一种打包。
*args: 将所有额外的位置参数打包成一个元组。
**kwargs: 将所有额外的关键字参数打包成一个字典。

#### 解包

```python
# 解包元组
person = ("Alice", 25)
name, age = person
print(name) # 输出: Alice
print(age)  # 输出: 25

# 解包列表
fruits = ["apple", "banana", "cherry"]
f1, f2, f3 = fruits
print(f1) # 输出: apple

# 解包字符串
a, b, c = "XYZ"
print(a) # 输出: X
```

- 基本解包
这是最常见的形式，要求变量的数量必须与容器中元素的数量完全一致。

```python
# 收集中间的元素
numbers = [1, 2, 3, 4, 5]
first, *middle, last = numbers
print(first)  # 输出: 1
print(middle) # 输出: [2, 3, 4]
print(last)   # 输出: 5

# 收集末尾的元素
a, *b = [1, 2, 3]
print(a) # 1
print(b) # [2, 3]

# 收集开头的元素
*a, b = [1, 2, 3]
print(a) # [1, 2]
print(b) # 3
```

- 扩展解包 (使用 *)
当变量数量与元素数量不一致时，可以使用* 来收集“剩余”的元素。这使得解包非常灵活。

```python
def introduce(name, age, city):
    print(f"我叫{name}，今年{age}岁，来自{city}。")

# 解包列表
person_info = ["小明", 18, "北京"]
introduce(*person_info) # 等价于 introduce("小明", 18, "北京")

# 解包字典
person_dict = {"name": "小红", "age": 20, "city": "上海"}
introduce(**person_dict) # 等价于 introduce(name="小红", age=20, city="上海")
```

- 在调用函数时，*和 **用于将一个序列或字典“拆开”作为参数传入。
*iterable: 将列表或元组解包成独立的位置参数。
**dict: 将字典解包成独立的关键字参数。

## set

无序且不重复的元素序列，可使用花括号 {}创建：将元素用逗号分隔，放在花括号内。也可以使用 set() 构造函数创建：可以从一个可迭代对象（如列表、元组、字符串）创建集合，特别注意：创建一个空集合必须使用 set()，而不是 {}。因为 {} 在 Python 中是用来创建一个空字典的。
有元素唯一性 (去重)：集合会自动剔除重复的元素
无序性：集合中的元素没有固定的顺序。这意味着不能通过索引来访问元素，每次打印集合时，元素的显示顺序也可能不同。
元素必须是可哈希的 (immutable)：集合中的元素必须是不可变类型，例如数字、字符串、元组。列表、字典等可变类型不能作为集合的元素。

### set常用函数

#### set添加元素

- s.add(x): 向集合 s 中添加单个元素 x。如果元素已存在，则不执行任何操作。
- s.update(iterable): 将一个或多个可迭代对象（如列表、集合）中的元素添加到集合 s 中。

#### set删除元素

- s.remove(x): 从集合 s 中移除元素 x。如果元素不存在，会抛出 KeyError 错误。
- s.discard(x): 从集合 s 中移除元素 x。如果元素不存在，不会抛出错误。
- s.pop(): 无参数，随机移除并返回集合中的一个元素。由于集合是无序的，无法确定会移除哪一个。
- s.clear(): 清空整个集合。

#### 数学运算

```py
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a | b) # 并集: {1, 2, 3, 4, 5, 6}
print(a & b) # 交集: {3, 4}
print(a - b) # 差集: {1, 2}
print(a ^ b) # 对称差集: {1, 2, 5, 6}
```

## dict

数据以 键: 值 的形式成对出现。键的唯一性：同一个字典中，键（key）必须是唯一的。如果尝试添加一个已存在的键，新的值会覆盖旧的值。键必须是“可哈希的”（hashable），通常指不可变类型，如字符串（str）、数字（int, float）或元组（tuple）。列表（list）或另一个字典（dict）不能作为键。高效查找：字典的查找速度非常快，无论数据量多大，查找时间基本恒定（时间复杂度为 O(1)），因为它使用哈希表实现。在 Python 3.7 及更高版本中，字典会记住键值对的插入顺序。可以使用花括号 {} 创建，也可以使用 dict() 构造函数，适合从其他可迭代对象（如包含元组的列表）创建字典。

### 遍历

#### 遍历键值对

方法：使用 .items()

```py
student = {'name': 'Alice', 'age': 25, 'city': 'New York'}

# 直接解包成 key 和 value
for key, value in student.items():
    print(f"{key}: {value}")
```

#### 遍历键

方法：直接遍历字典，或使用 .keys()

```python
# 方式一：直接遍历（推荐，简洁）

for key in student:
    print(key)

# 方式二：显式调用 .keys()（语义更明确）

for key in student.keys():
    print(key)
```

#### 遍历值

方法：使用 .values()

```python
for value in student.values():
    print(value)
```

### 字典推导式

这是一种简洁高效的方式，适合根据某种规律或从其他数据生成字典。基本结构:{键表达式: 值表达式 for 变量 in 可迭代对象}

| 场景 | 语法结构示例 | 说明 |
| :--- | :--- | :--- |
| 纯转换 | `{k: v*2 for k, v in data.items()}` | 对每个值进行转换 |
| 纯筛选 | `{k: v for k, v in data.items() if v > 0}` | 只保留满足条件的键值对 |
| 转换+筛选 | `{k: v*2 for k, v in data.items() if v > 0}` | 先筛选再对值进行转换 |
| 键值互换 | `{v: k for k, v in data.items()}` | 将键和值互换 |
| 值分支 | `{k: (v if v>0 else 0) for k, v in data.items()}` | 最常见：键不变，值根据条件变化 |
| 键分支 | `{(k if k>0 else 'neg'): v for k, v in data.items()}` | 值不变，键根据条件变化 |
| 全分支 | `{(k if k>0 else 'neg'): (v if v>0 else 0) for k, v in data.items()}` | 键和值同时根据条件变化 |

### dict常用函数

#### 增、改操作

- d[key] = value: 向字典 d 中添加或修改一个键值对。如果键已存在，则更新其对应的值。
- d.update(other_dict): 将另一个字典 other_dict 中的所有键值对批量更新到字典 d 中。

#### 查操作

- d[key]: 通过键直接访问对应的值。如果键不存在，会抛出 KeyError 错误。
- d.get(key, default): 安全地获取键对应的值。如果键不存在，则返回 None 或你指定的默认值 default。
- d.keys(): 返回一个包含字典中所有键的动态视图。
- d.values(): 返回一个包含字典中所有值的动态视图。
- d.items(): 返回一个包含所有 (键, 值) 元组的动态视图，常用于遍历。

#### 删操作

- d.pop(key): 删除指定的键值对，并返回被删除的值。
- d.popitem(): 删除并返回字典中最后一个插入的键值对。
- del d[key]: 直接删除指定的键值对，不返回任何值。
- d.clear(): 清空字典中的所有元素。

#### dict其他常用操作

- d.copy(): 创建并返回字典 d 的一个浅拷贝副本。
- len(d): 返回字典中键值对的数量。
- key in d: 判断一个键是否存在于字典中，返回 True 或 False。
