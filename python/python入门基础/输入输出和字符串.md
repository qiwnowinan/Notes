# python入门基础语法

## 格式化输出

```python
name = "Alice"
age = 25
print(f"大家好，我叫{name}，今年{age}岁。")

x, y = 10, 20
print(f"{x} + {y} = {x + y}")
# 输出: 10 + 20 = 30
```

### 数字格式化

```python
pi = 3.1415926
money = 1234567890
ratio = 0.756

print(f"圆周率约等于: {pi:.2f}")      # 输出: 圆周率约等于: 3.14   .2表示保留为小数
print(f"金额: {money:,} 元")        # 输出: 金额: 1,234,567,890 元   ,表示添加逗号分隔符
print(f"完成率: {ratio:.1%}")       # 输出: 完成率: 75.6%   .1表示1位小数  %表示百分比形式
```

### 文本对齐与填充

```python
text = "Python"
print(f"|{text:<10}|")   # 左对齐，总宽10: |Python    |
print(f"|{text:>10}|")   # 右对齐，总宽10: |    Python|
print(f"|{text:^10}|")   # 居中对齐，总宽10: |  Python  |
print(f"|{text:#^10}|")  # 居中对齐，用#填充: |##Python##|
```

## 格式化输入

python没有专门格式化输出的函数，通过 input() + 字符串处理 + map() 实现类似功能  
`a = int(input())` 用类似的方法输入，因为input得到的是string,所以要int()进行类型转换  
`a, b = map(int, input().split()) # 按空格分割并转为整数` 输入多个值
`nums = list(map(int,input().split()))  # 在一行内读取多个由空格分隔的数字，并将它们直接转换成一个整数列表。`

### map函数

map() 用于对可迭代对象的每个元素应用一个函数。
语法：map(function, iterable)
返回：一个迭代器（惰性求值，节省内存）。
function: 要执行的函数（可以是内置函数、自定义函数或 lambda）。
iterable: 一个或多个可迭代对象

```python
# 假设输入: 10 20
a, b = map(int, input().split()) 
print(a + b) # 输出: 30
```

### 常用函数

#### 查找与判断

- s.find(sub)：查找子字符串 sub 在 s 中第一次出现的位置（索引），如果找不到则返回 -1。
- s.index(sub)：与 find() 类似，但如果找不到子字符串会抛出 ValueError 异常。
- s.count(sub)：统计子字符串 sub 在 s 中出现的次数。
- sub in s：判断字符串 s 中是否包含子字符串 sub，返回布尔值

#### 拆分与组合

- s.split(sep)：使用指定的分隔符 sep 将字符串分割成一个列表。如果不指定 sep，默认按空白字符（空格、换行等）分割。
- sep.join(iterable)：将可迭代对象（如列表、元组）中的字符串元素用 sep 连接成一个新的字符串。
- s.replace(old, new)：将字符串中的旧子字符串 old 替换为新子字符串 new。
- s.strip()：移除字符串头尾的空白字符（包括空格、换行符 \n、制表符 \t 等）。
- s.lstrip() / s.rstrip()：仅移除字符串左侧（开头）或右侧（结尾）的空白字符。

#### 大小写转换与调整

- s.lower()：将字符串中的所有字符转换为小写。
- s.upper()：将字符串中的所有字符转换为大写。
- s.capitalize()：将字符串的第一个字符转换为大写，其余转换为小写。
- s.title()：将字符串中每个单词的首字母转换为大写。

#### 内容检测

- s.isalpha()：如果字符串非空且所有字符都是字母，则返回 True。
- s.isdigit()：如果字符串非空且所有字符都是数字，则返回 True。
- s.isalnum()：如果字符串非空且所有字符都是字母或数字，则返回 True。
