# 函数

先分清函数与方法

| 特性 | 内置函数 (如 `print`) | 方法 (如 `string.upper`) |
| :--- | :--- | :--- |
| 归属 | 独立的，不属于任何对象 | 属于特定的对象（如字符串、列表） |
| 调用格式 | `函数名(参数)` | `对象.方法名(参数)` |
| 类比 | 工具箱里独立的工具 | 设备上专用的按钮 |
| 更多例子 | `len()`, `type()`, `input()` | `list.append()`, `str.lower()`, `str.replace()` |

## 定义

在 Python 中，我们使用 def 关键字来定义一个函数。
基本语法：

```py
def 函数名(参数列表):
    """函数的文档字符串，用于说明函数功能（可选但推荐）"""
    # 函数体：实现具体功能的代码
    # 注意：函数体需要缩进（通常是4个空格）
    return 返回值  # 返回值是可选的
```

## 参数

- 位置参数
调用函数时，需要按照参数定义的顺序依次传入值。

```python
def subtract(a, b):
    return a - b

print(subtract(10, 5))  # 输出: 5
print(subtract(5, 10))  # 输出: -5
```

- 关键字参数
调用函数时，通过 参数名=值 的形式指定参数，这样可以不按照顺序传参。

```python
print(subtract(b=5, a=10))  # 输出: 5
```

- 默认参数
在定义函数时为参数提供一个默认值。如果调用时没有传入该参数，则使用默认值。

```python
def power(base, exp=2):
    """计算 base 的 exp 次方，exp 默认为 2"""
    return base ** exp

print(power(3))    # 输出: 9 (使用默认的 exp=2)
print(power(3, 3)) # 输出: 27
```

- 可变参数
当不确定需要传入多少个参数时，可以使用可变参数。
*args: 接收任意数量的位置参数，并将它们打包成一个元组（tuple）。
**kwargs: 接收任意数量的关键字参数，并将它们打包成一个字典（dict）。

```python
def sum_all(*args):
    """计算任意个数字的和"""
    return sum(args)

print(sum_all(1, 2, 3))       # 输出: 6
print(sum_all(1, 2, 3, 4, 5)) # 输出: 15

def print_info(**kwargs):
    """打印任意个关键字信息"""
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=25, city="Beijing")

# 输出
# name: Alice
# age: 25
# city: Beijing
```

## 返回值

函数使用 return 语句来返回结果。return 会终止函数的执行，并将结果返回给调用者。

- 返回单个值
- 返回多个值，Python 函数可以返回多个值，这些值会被打包成一个元组。
- 无返回值，如果函数中没有 return 语句，或者只写 return，函数会默认返回 None。
